# 🎮 squid-game-ui

A cinematic Squid Game inspired landing page built using HTML & CSS.

This project was created while practicing modern frontend layout structuring, Flexbox, CSS positioning, gradients, and glassmorphism inspired UI design concepts.

---

## 🚀 Features

- Squid Game Inspired UI
- Cinematic Neon Aesthetic
- Glassmorphism Thumbnail Section
- Gradient Background Effects
- Modern Hero Layout
- Interactive Buttons
- Social Sidebar
- Hover Effects & Visual Depth

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Positioning
- Font Awesome
- Google Fonts

---

## 📚 Concepts Practiced

- display: flex
- position: relative
- position: absolute
- UI Structuring
- Hero Section Design
- Alignment & Spacing
- Layering & Visual Hierarchy
- Glassmorphism Effects
- Gradient Composition
